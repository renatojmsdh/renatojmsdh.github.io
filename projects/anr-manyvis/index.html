MANYVIS: Learning physics-aware video descriptions 

"MANYVIS: Learning physics-aware video descriptions" is a JCJC ANR project funded by the "ANR CES 23 – Artificial Intelligence" since october 2023 for a duration of 48 months.

Goals of the project

The main objectives of this project is to develop new image processing and computer vision algorithms taking into account the manifold structure and temporal constraints into account.
